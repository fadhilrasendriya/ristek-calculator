# Ristek Calculator

### Description / Overview
This app will do mathematical operations (addition, substraction, multiplication, and division) on two number inputs

### Requirements
- gradle version 5.6.3
- java 12

### Installation
1. Clone this repo
2.
```
./gradlew build
```

### How to run
```
./gradlew run
```
